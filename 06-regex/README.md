# Let's build regex engine from scratch

## Introduction
In this project, we will build a regex engine from scratch. We will implement the following regex operators:
- Literal: `a`, `Z`, `5` , `#`, `&` etc.
- Wildcard: `.`
- Group: `( )`
- Excaped : `\x`
- Optional: `?`
- Zero or more: `*`
